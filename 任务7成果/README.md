```
│  UserService | AuthService          │
│  RoleService | PermissionService    │
├─────────────────────────────────────┤
│            数据访问层                │  ← 基础服务和连接管理
│        BaseService | 数据库连接      │
├─────────────────────────────────────┤
│            数据模型层                │  ← 8个数据模型
│  User | Role | Permission | 关联表   │
└─────────────────────────────────────┘
```

#### Windows用户
```bash

start_swagger.bat
```


### 📍 文档访问地址

启动成功后，访问以下地址查看API文档：

- **Swagger UI**: http://localhost:8000/docs (推荐用于截图)

- **OpenAPI JSON**: http://localhost:8000/openapi.json (原始规范)

- **角色管理**: 4个接口 (创建、列表、详情、权限分配)
